# How to Crete Presets

For create presets you need a `.json` file in `/presets` with `ID` and any other parameter who you want. 

Note that not all pypresence parameters work in Sylo.

## Preset Example
````json
{
    "ID": "An app ID from Discord Developers Portal",
    "name": "Sylo",
    "details": "I'm using Sylo"
}
````
